Description

The game involves piloting a spacecraft in an arena where enemies launch missiles at the player. These missiles explode if the planes firing them are hit; after three missile hits, the planes themselves explode. Additionally, there is an invincible turret that also fires missiles. The parallax effect on the background adds a fun and engaging layer of depth. Graphics were also created for the Google Play Store page, featuring a space station.

Features

There are portals in the four different directions. The portals are divided into four different sections, each linking to a different portal. The enemies follow a certain flight path that they copy from the player.

Challenges

The biggest hurdles were in making the missiles track the player fairly and not destroy the player too fast. Also, building the game APK was quite complicated and worked randomly. The game balancing wasn’t that bad; it caused quite a bit of difficulty because the missiles are infinitely replicating. The game has an interesting quirk, which is that if you have it run on 30 fps, it looks really jarring. The solution was to make it run at 60 fps on phones, which was done through code.

Self reflection

In hindsight, the project lacked a bit of direction and was sort of finished and sort of not, which was apparent later in a few tests. The main problem with the game was that it didn't communicate to the player enough. In the projects that I have done since, I have thought more about how to communicate to the player about different gameplay mechanics.

The player didn’t have any way, for instance, to know that aircraft can be damaged with the missiles because the first impact didn’t show any damage to the plane. As for my part as a coder and an artist, I think I did a passable job on the assets. There is a lot of improvement to be had, but it was also my first try at drawing for a game.
